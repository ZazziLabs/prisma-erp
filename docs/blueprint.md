# **App Name**: Prisma ERP

## Core Features:

- Entrada de Caixa: Permite registrar entradas de dinheiro especificando a fonte e o valor.
- Saída de Caixa: Permite registrar saídas de dinheiro especificando a finalidade e o valor.
- Organização diária: Lista todas as transações do dia, mostrando o saldo final.
- Cálculo de Saldo: Calcula automaticamente o saldo atual com base nas entradas e saídas.
- Resumos e Relatórios: Gera resumos diários, semanais e mensais das movimentações de caixa para análise.
- Alertas Inteligentes: Usa IA para monitorar os padrões de fluxo de caixa e envia alertas quando o fluxo se desvia dos padrões esperados, como um aviso antecipado sobre a necessidade de restringir gastos.

## Style Guidelines:

- Cor primária: Azul moderado (#5DADE2) para refletir segurança e organização financeira.
- Cor de fundo: Azul muito claro (#EBF5FB), quase branco.
- Cor de destaque: Verde (#A9DFBF), para indicar entradas positivas e sucesso financeiro.
- Fonte para corpo do texto: 'Inter', uma fonte sans-serif moderna e objetiva para facilitar a leitura.
- Fonte para cabeçalhos: 'Space Grotesk', uma fonte sans-serif com um toque moderno e tecnológico.
- Ícones claros e intuitivos para representar entradas, saídas e categorias financeiras.
- Layout limpo e organizado, com foco na facilidade de uso e visualização rápida das informações financeiras.
- Transições suaves ao navegar entre as diferentes seções e ao registrar novas transações.
